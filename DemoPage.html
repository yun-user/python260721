HTML5 · CSS3 · JavaScript

한 파일로 배우는 데모 페이지

이 페이지는 HTML5 구조, CSS3 레이아웃과 스타일, JavaScript 기본 동작을 한 번에 살펴볼 수 있도록 만든 연습용 예제입니다. 버튼을 눌러 동작을 확인하고, 아래 코드 박스를 보면서 각 영역의 역할을 익혀보세요.

HTML5

header, main, section, article, footer 같은 시맨틱 태그를 사용해 구조를 나눕니다.

CSS3

그라데이션 배경, 카드 UI, 반응형 그리드, 그림자, 둥근 모서리 등 자주 쓰는 스타일을 포함했습니다.

JavaScript

버튼 클릭 이벤트로 텍스트를 바꾸고, 현재 시간을 화면에 표시하는 간단한 기능을 넣었습니다.

실습 영역

버튼을 눌러 페이지의 안내 문구를 바꿔보세요. 스스로 코드를 수정하면서 구조와 스타일을 분리해 보는 연습에 좋습니다.

여기에 JavaScript 결과가 표시됩니다.
<section> <h2>실습 영역</h2> <button>메시지 바꾸기</button> </section>
